Разработка
===========

.. important::

    Eсли система поднята локально без eis (keycloak), то используется **BASIC_AUTH_ACCESS** и при добавлении пользователя его нужно добавить в :ref:`docker-файл<ecos_proxy_app>`

Раздел предназначен для разработчиков, работающих с платформой Citeck. В нем собраны материалы по ключевым механизмам платформы: записям и атрибутам, API, артефактам, аспектам, предикатам, транзакциям и событиям. Отдельные статьи описывают работу микросервисов, настройку интеграций, локализацию на стороне сервера, а также инструменты для ускорения разработки.
