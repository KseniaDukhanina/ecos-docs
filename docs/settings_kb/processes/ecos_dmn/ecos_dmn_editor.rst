Моделирование принятия решения
============================================================

Редактор DMN построен на базе библиотеки `dmn-js <https://bpmn.io/toolkit/dmn-js/>`_ и предоставляет визуальную среду для моделирования логики принятия решений без написания кода. С его помощью можно создавать схемы из стандартных элементов нотации DMN, настраивать таблицы решений с условиями и выходными значениями, а затем публиковать готовые модели в движок для использования в бизнес-процессах.

Раздел охватывает:

- **Конструктор DMN** — интерфейс редактора, панели инструментов и работа с элементами схемы;
- **Компоненты Citeck DMN** — типы элементов нотации и правила их соединения.
