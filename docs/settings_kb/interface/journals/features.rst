.. _journal_features:

Функции
=========

Раздел описывает дополнительные возможности журналов платформы Citeck — от управления данными и настройки отображения до автоматизации и вычислений на основе атрибутов.

Раздел охватывает:

* :ref:`Групповые действия <group_actions_journal>` — настройка действий, применяемых одновременно к нескольким записям
* :ref:`Вычисляемые атрибуты <computed_attributes>` — вычисление значений столбцов на основе данных из базы данных
* :ref:`Выражения <journal_expressions>` — операторы и функции для формирования значений в столбцах
* :ref:`Форматтеры <formatters>` — управление отображением значений в ячейках
* :ref:`Редакторы <editors>` — встроенное редактирование значений непосредственно в таблице
* :ref:`Синхронизация атрибутов <attribute_synchro>` — фильтрация задач по атрибутам связанного документа
