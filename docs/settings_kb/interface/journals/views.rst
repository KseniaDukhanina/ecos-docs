Режимы отображения
====================

Помимо стандартного табличного представления, журналы платформы Citeck поддерживают несколько альтернативных режимов отображения данных — для работы с задачами, файлами и контентом в удобном для пользователя виде.

Раздел охватывает:

* :ref:`Канбан-доска <kanban_board>` — представление записей в виде карточек, сгруппированных по статусам или другим атрибутам
* :ref:`Библиотека документов <document_library>` — иерархический интерфейс для работы с папками и файлами
* :ref:`Список и плитки <tiles>` — альтернативные режимы отображения записей в виде плиток или компактного списка
