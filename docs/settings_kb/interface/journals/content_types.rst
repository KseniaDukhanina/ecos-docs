Типы контента
================

Платформа Citeck поддерживает специализированные типы контента, расширяющие возможности журналов для работы с публикациями, новостями и внутренней документацией.

Раздел охватывает:

* :ref:`Публикация <publication>` — управление контентом с поддержкой черновиков, версионирования и публикации записей
* :ref:`Новости <news>` — виджет и журнал для публикации анонсов в рабочих пространствах
* :ref:`База знаний <wiki_base>` — ведение внутренней базы знаний в рабочих пространствах
