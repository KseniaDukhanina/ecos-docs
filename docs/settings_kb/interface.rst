Интерфейс
=========

Раздел охватывает настройку всех ключевых элементов интерфейса Citeck:

- **Формы** — графическое представление объектов в виде набора полей для ввода и редактирования данных. Раздел описывает конструктор форм, компоненты, локализацию, черновики и связь форм с задачами процесса.
- **Журналы** — табличное представление объектов с фильтрацией, группировкой и действиями над записями. Раздел охватывает создание журналов, режимы отображения (таблица, канбан, плитки, библиотека документов), групповые действия и вычисляемые атрибуты.
- **Меню** — навигационная панель Citeck с поддержкой гибкой настройки под группы пользователей. Раздел описывает структуру конфигурации, типы элементов, приоритеты групп и решение типичных проблем.
- **Дашборды** — настраиваемые домашние страницы пользователя или страницы кейса с поддержкой размещения и конфигурации виджетов.
- **Виджеты** — блоки отображения информации на дашбордах: документы, задачи, процессы, пользователи и другие данные системы.
- **Темы интерфейса** — визуальная кастомизация Citeck: загрузка и применение пользовательских логотипов, favicon и CSS-стилей в виде zip-архива.
