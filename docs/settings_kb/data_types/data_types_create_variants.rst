Варианты создания
==================

**Вариант создания** — это настраиваемый сценарий, по которому пользователь создаёт новый объект данного типа. Каждый вариант задаёт форму, доступность для конкретных ролей и поведение системы после создания записи.

Если для типа данных задано несколько вариантов создания, при нажатии кнопки **«+»** в журнале пользователь увидит список и сможет выбрать нужный. Если вариант только один, форма открывается сразу.

Настройка вариантов создания производится на вкладке :guilabel:`Варианты создания`.

Ограничение прав на создание
------------------------------

Создание можно ограничить через настройку вариантов создания. **"+"** в журнале и пункт в меню **"Создать"** появляются только, если для пользователя есть доступные варианты создания.

В настройке типа можно отключить вариант создания по умолчанию и добавить новый с указанием групп, которые будут иметь к нему доступ. Заполнение полей «Имя» и «Форма» необязательно — если оставить их пустыми, значения вычислятся автоматически.
