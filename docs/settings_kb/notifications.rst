Уведомления
===========

Раздел описывает настройку и использование уведомлений в Citeck.

Подразделы охватывают следующие темы:

- **Уведомления** — общий обзор механизма уведомлений: типы, каналы доставки и базовая конфигурация.
- **Шаблоны уведомлений** — создание и управление шаблонами для автоматической генерации текста писем и сообщений.
- **Массовая рассылка** — отправка уведомлений сразу нескольким получателям по заданным критериям.
- **Функционал Lazy approval для задачи** — согласование задач непосредственно через ответ на письмо, без входа в систему.
- **Отправка уведомлений из BPMN (Send Task)** — интеграция уведомлений в бизнес-процессы с помощью элемента Send Task.
- **Отправители** — настройка почтовых отправителей и параметров SMTP-подключения.
- **Отправка электронных писем, подтверждённых ЭЦП** — подписание исходящих писем электронной цифровой подписью.
- **Отправка тестового сообщения при запуске микросервиса** — проверка работоспособности почтового канала при старте сервиса.
