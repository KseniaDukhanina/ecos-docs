Мониторинг, производительность и аудит
========================================

Раздел охватывает инструменты наблюдения за состоянием платформы Citeck, анализа производительности и аудита действий пользователей.

- **Spring Boot Admin** — административный интерфейс для мониторинга микросервисов: состояние, метрики, логи и конфигурация каждого сервиса в реальном времени.
- **Аудит** — журналирование пользовательских действий и системных событий (Enterprise).
- **Производительность** — рекомендации по работе с вкладками и другие настройки, влияющие на отзывчивость интерфейса.
- **Нагрузочное тестирование** — результаты стресс-тестов платформы при высоких нагрузках.
