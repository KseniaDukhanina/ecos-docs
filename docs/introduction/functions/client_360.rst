Клиент 360
============

Стратегия **Клиент 360** — подход к работе с контрагентами, при котором вся информация о клиенте собрана в едином месте. В карточке контрагента отображаются консолидированные данные из всех модулей платформы: CRM, договоры, документы, обращения в поддержку, корреспонденция и проекты.

Такая консолидация позволяет:

- получать полную картину взаимодействия с клиентом без переключения между разделами;
- оперативно анализировать историю сделок, платежей и обращений;
- принимать обоснованные решения на основе актуальных данных.

Настройки
----------

DaData
~~~~~~~

Получение токена
""""""""""""""""""

1. Перейдите на https://dadata.ru и нажмите кнопку **«Войти»** в правом верхнем углу экрана.

2. В открывшемся окне выберите **«Зарегистрироваться»**. Введите имя пользователя, вашу электронную почту и пароль, затем нажмите кнопку регистрации. 

3. Подтвердите email. На указанный почтовый ящик придёт письмо со ссылкой для подтверждения — нажмите на неё.  Без этого шага ключи не будут сгенерированы.

4. Получите ключи в личном кабинете. После подтверждения email API-ключ автоматически сгенерируется.  Перейдите в личный кабинет по адресу https://dadata.ru/profile/ — в нижней части страницы будут указаны API-ключ и секретный ключ вашего аккаунта. 

5. Скопируйте нужные ключи. В DaData используются два ключа:

 - **API-ключ** — основной пароль от сервиса, который указывается в запросах к DaData по API.  Используется для подсказок (Suggestions) и большинства операций.
 - **Секретный ключ** — дополнительный ключ, который нужен для API стандартизации (Cleaner).

Скопируйте оба ключа и вставьте их в настройки вашего приложения или используйте в коде.

Контур Фокус
~~~~~~~~~~~~~

Получение токена
""""""""""""""""""

1. Через личный кабинет Контур.Фокус

 - На https://focus.kontur.ru/ перейдите в раздел **«Настройки»** или **«API»**
 - Найдите пункт **«Ключ доступа»** / **«API-ключ»**
 - Скопируйте или сгенерируйте токен

2. Через учётную запись Контур (SSO)

 - На https://kontur.ru/ войдите в свой аккаунт
 - В разделе интеграций или для разработчиков найдите доступ к API Фокуса
 - Токен выдаётся в рамках подписки на соответствующий тариф

Настройка Citeck
""""""""""""""""""
