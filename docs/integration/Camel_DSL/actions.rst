Действия, доступные с Camel DSL
================================

Для каждого артефакта Camel DSL в журнале доступны действия управления контекстом. Помимо стандартных операций (скачать, редактировать, удалить), предусмотрены дополнительные действия, специфичные для Camel DSL: запуск и остановка контекста, а также просмотр лога последней синхронизации.

.. image:: _static/actions/camel_actions.png
   :width: 700
   :align: center

|

Описание дополнительных действий:

.. list-table::
   :widths: 10 90
   :header-rows: 1

   * - Действие
     - Описание
   * - **1a** — Старт
     - Запускает Camel-контекст. Доступно, когда контекст остановлен.
   * - **1b** — Стоп
     - Останавливает выполняющийся Camel-контекст.
   * - **2** — Лог
     - Открывает журнал последней синхронизации с подробной информацией о ходе выполнения и возможных ошибках.

Кнопки **Старт** и **Стоп** также доступны непосредственно внутри карточки артефакта:

.. image:: _static/actions/action_start.png
   :width: 600
   :align: center

|

.. note::
   Контекст автоматически останавливается сразу после завершения всех задач — например, при импорте данных из файла. Это ожидаемое поведение: после обработки файла контекст переходит в состояние **STOPPED**.

   Подробнее о возможных состояниях контекста см. в разделе :ref:`Инстансы контекста Camel DSL <camel_instance>`.
