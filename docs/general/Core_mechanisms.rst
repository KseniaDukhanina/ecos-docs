Механизмы платформы
=====================

В данном разделе описаны ключевые механизмы платформы Citeck, обеспечивающие взаимодействие компонентов, интеграцию с внешними системами и надёжную работу бизнес-логики.

Раздел включает следующие темы:

- **Команды** — механизм декларативного вызова действий над записями платформы.
- **События** — система публикации и подписки на события внутри платформы.
- **Citeck WebAPI** — REST-интерфейс для работы с данными и сервисами платформы.
- **Транзакции** — управление транзакционностью операций на основе двухфазного коммита.
- **Внешние коммуникации** — настройка каналов уведомлений (email, Telegram и др.).
