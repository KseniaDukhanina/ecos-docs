.. _developer_tools:

Инструменты разработчика
==========================

Раздел содержит описание инструментов, предназначенных для разработки и отладки на платформе Citeck.

- :ref:`Консоль разработчика <developer_console>` — встроенный инструмент для выполнения JavaScript-кода прямо в браузере с отображением результатов в реальном времени. Доступна администраторам из раздела администрирования.

- :ref:`Плагин для IntelliJ IDEA <IntelliJ_IDEA_plugin>` — расширение для IDE, которое ускоряет работу с проектами и артефактами платформы: поддерживает автодополнение, деплой артефактов и навигацию по конфигурациям прямо из редактора.
