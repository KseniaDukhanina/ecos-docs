Работа с данными
=================

Раздел описывает инструменты и API платформы Citeck для работы с данными: получение, запись, фильтрация и расширение атрибутов сущностей.

- :ref:`Records API <Records_API>` — единый интерфейс для доступа к данным системы из браузера, мобильного приложения, микросервисов и других потребителей;
- :ref:`Миксины <mixins>` — механизм расширения атрибутивного состава записей из любой части системы;
- :ref:`Язык предикатов <ecos-predicate_main>` — язык запросов для фильтрации и поиска записей на стороне frontend и backend;
- :ref:`Системные атрибуты <system_attributes>` — зарезервированные атрибуты платформы, доступные для любой сущности;
- :ref:`Citeck Data <ecos_data_main>` — библиотека для создания Records DAO с хранением данных в БД.
