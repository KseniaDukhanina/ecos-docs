Оффлайн-режим
============================

При отсутствии интернета действия, выполненные с задачей, сохраняются в Local Storage и отправляются на сервер при возобновлении связи.

Загруженные задачи не сохраняются в какую-либо базу данных или Local Storage и доступны, пока приложение не будет выгружено из памяти.

Согласование задачи
--------------------------------------

``src/store/actions/forms.actions.ts approveFormWorker``

* при согласовании задачи проверяется наличие интернета; в случае его отсутствия действие записывается в **Local Storage**
* сохранённые действия будут отправлены на сервер при согласовании следующей задачи при наличии доступа в интернет **(loadActionsWorker)**
  ``src/store/actions/offline-actions.actions.ts loadActionsWorker``
